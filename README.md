# 健康饮食记录 App

这是一个基于 HTML5, CSS3 和 JavaScript 的移动端网页应用，用于记录每日饮食、运动、体重和饮水情况。

## 功能特点

1.  **热量管理**：设置每日目标热量，自动计算剩余热量（目标 - 摄入 + 运动）。
2.  **饮食记录**：记录每顿饭的热量。
3.  **运动记录**：记录运动项目和消耗热量。
4.  **饮水打卡**：
    *   快捷记录：250ml, 500ml。
    *   **自定义记录**：输入任意毫升数添加。
5.  **体重追踪**：记录每日体重。
6.  **日期切换**：
    *   支持左右切换日期。
    *   **日历选择**：点击日期显示区域，可直接选择年、月、日。
7.  **数据持久化**：所有数据保存在浏览器的本地存储（LocalStorage）中。

## 界面设计

*   **风格**：现代简约风格，采用深青色（Deep Teal）与金色搭配，视觉体验成熟大方。
*   **交互**：针对移动端优化的触控区域和反馈。

## 如何使用

1.  直接双击打开 `index.html` 文件即可在浏览器中运行。
2.  推荐使用 Chrome 或 Safari 浏览器的移动端模式（F12 -> 切换设备工具栏）以获得最佳体验，或者直接在手机上打开。

## 文件结构

-   `index.html`: 页面结构
-   `styles.css`: 样式表
-   `script.js`: 业务逻辑

## 技术栈

-   原生 HTML/CSS/JavaScript
-   FontAwesome (图标)
-   Google Fonts (字体)
